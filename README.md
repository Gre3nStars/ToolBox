# ToolBox ver 1.0
这是由PySide6开发的一个集编解码、普通加解密以及常用小工具于一体的工具箱，只为日常工作使用，工具的实现大部分由AI生成编写。


Q：为什么写这个工具，市面上有很多类似的工具了，如编解码cyberchelf非常强大
A：因为使用习惯问题，及一些工具使用频次情况，还有就是页面切换使用问题，当我处理一个编解码或加解密等时，我希望互不干扰，同时使用又方便切换！
